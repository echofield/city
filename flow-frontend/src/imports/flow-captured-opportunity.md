Good — this is where Flow can become **addictive in a healthy way for drivers**.

The mechanic is called **“Captured Opportunity” feedback**.

Right now Flow tells drivers:

* where to go
* what is forming
* expected € / hour

That’s good.

But what makes tools like this powerful is when the driver **feels the win came from the system**.

You want drivers to think:

> “Flow just made me money.”

Not just:

> “Flow gave information.”

---

# The mechanic: Opportunity Capture

After a signal is active, the system tracks a **possible outcome window**.

Example:

Driver sees:

```
ANTICIPE
BASTILLE
€15–40
PIC 02:00
```

They go there.

After the wave passes, Flow shows:

```
OPPORTUNITY CAPTURED
€22–37 potential window
```

or

```
WAVE RESULT
Strong
```

Even if the system only estimates.

---

# Why this is powerful

The driver’s brain connects:

```
Flow signal → ride → money
```

Instead of:

```
random ride
```

You’re reinforcing the **cause–effect loop**.

That creates habit.

---

# How it looks in UI

Very subtle, small line in LIVE or TRACE.

Example:

```
TRACE
02:14 — ANTICIPE Bastille
RESULT: STRONG WINDOW
```

Or:

```
RESULT
+€28 opportunity window
```

Doesn’t need to be perfect.

Just **credible**.

---

# This creates the trading effect

Traders check tools constantly because they see:

```
signal
entry
result
```

Flow becomes similar:

```
signal
position
window
```

Drivers start thinking:

> “Let’s see what the next one does.”

# Even better: streak feedback

Example small indicator:

```
LAST 3 SIGNALS
✓ strong
✓ medium
✓ strong
```

Or

```
3 / 4 windows successful
```

Now the driver thinks:

> “Flow is on fire tonight.”

Even if it's probabilistic.

---

# The loop becomes

1️⃣ Driver opens Flow
2️⃣ Sees signal
3️⃣ Moves
4️⃣ Gets ride
5️⃣ Flow later shows **window result**

This creates **memory reinforcement**.

Never say:

```
YOU EARNED
```

Say:

```
OPPORTUNITY WINDOW
```

Because Flow doesn’t control the ride.

But the brain still connects it.
