# The problem

Right now drivers open Flow when they **think something might happen**.

Example behavior:

```
open app
look
nothing new
close
```

Maybe they open it again **20–30 minutes later**.

But if they forget to check during the **formation phase**, they miss the opportunity.

---

# The solution: the City Pulse

Add a **small live pulse indicator** in the header.

Example:

```
FLOW • PULSE
```

Then a **small moving bar or signal**.

Example:

```
CALME
MONTEE
FORMATION
PIC
DISPERSION
```

With a **moving dot**.

Example:

```
CALME — MONTEE — ● FORMATION — PIC — DISPERSION
```

The dot slowly moves as the system detects activity.

---

# Why this works psychologically

Drivers start thinking:

```
is the city waking up?
```

So they open Flow just to **check the pulse**.

Even when nothing big happens.

It becomes a **habit check**.

Exactly like traders checking markets.
