# 1 — Late International Train Arrivals (Gare du Nord)

This is one of the **best signals in Paris**.

Why?

Because international passengers have:

• luggage
• fatigue
• hotels
• weak knowledge of metro
• company budgets

Which means **high taxi conversion**.

### Core triggers

* Eurostar arrivals
* Thalys arrivals
* clustered arrival windows

Example window:

```text
00:35 – 01:10
Eurostar arrivals
```

### Conversion factors

Strong when:

• late hour
• rain
• metro reduced
• luggage heavy

### Ride profile

Usually:

```
€25 – €70
```

Destinations:

• hotels
• west Paris
• business districts

### Flow signal example

```
GARE DU NORD
Eurostar + Thalys release

► POSITION SOUTH EXIT

00:40 – 01:10

LONG RIDES
HOTELS / WEST PARIS
```

This is a **gold mine signal**.

```
long-haul arrivals
```

or

```
20:20 – 21:15
EU arrivals cluster
```

### Conversion multipliers

Higher when:

• luggage delay
• rain
• late evening
• metro weak

### Ride profile

Very strong.

```
€35 – €80
```

Sometimes:

```
€90+
```

### Flow signal

```
CDG T2E

20:45 – 21:30

LONG RIDES
HOTELS / WEST PARIS
```

Airport waves are **structural money**.

# 3 — Concert / Arena Exits (Accor Arena)

This is one of the **best urban bursts**.

Because:

• thousands exit simultaneously
• many miss metro
• rain multiplies demand

### Trigger pattern

Concert ends:

```
22:30
```

Conversion peak:

```
22:45 – 23:10
```

Most drivers arrive **too late**.

Flow can predict **positioning before exit**.

### Ride profile

Mixed but valuable.

```
€15 – €45
```

Often multiple rides quickly.

### Flow signal

```
ACCOR ARENA

Concert exit

► POSITION EAST BERCY

22:45 – 23:20

MIXED RIDES
HIGH FREQUENCY
```

# 4 — Nightlife Closure Cascade (Pigalle / Marais)

This is different.

Not a single release.

A **cascade of releases**.

### Pattern

Bars close progressively.

Example:

```
02:30 – 04:30
```

Strong zones:

• Pigalle
• Bastille
• Marais

### Conversion triggers

Very strong when:

• last metro passed
• rain
• weekend

### Ride profile

Many small rides but frequent.

```
€8 – €20
```

But **high ride density**.

### Flow signal

```
PIGALLE

Nightlife closure

► HUNT

03:10 – 04:20

SHORT RIDES
HIGH FREQUENCY
```
