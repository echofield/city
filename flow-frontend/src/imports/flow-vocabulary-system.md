Perfect. Let’s design a **tight vocabulary system for FLOW** — about **25 words max** — so the whole product speaks the **same language of the city**.

Goal:
The app should sound like **an experienced Paris driver whispering useful hints**, not like software.

Principles:

* **Short**
* **Concrete**
* **Observable**
* **Human**
* **No analytics language**

# 1. The 5 Core ACTION verbs (most important)

These appear in the big command line.

| Verb           | Meaning           |
| -------------- | ----------------- |
| **Maintenir**  | Stay in area      |
| **Rejoindre**  | Move to area      |
| **Anticiper**  | Go early          |
| **Contourner** | Avoid congestion  |
| **Tenter**     | Opportunistic try |

Example:

```
REJOINDRE
Montmartre
```

or

```
MAINTENIR
Quartier Latin
```

These feel **natural and professional**, not military.

# 5. Time hints

Drivers understand **approximate time**, not exact analytics.

Use:

| Phrase             | Meaning           |
| ------------------ | ----------------- |
| **Dans ~5 min**    | about to start    |
| **Maintenant**     | happening now     |
| **Encore ~10 min** | window still open |
| **Fini**           | done              |

Example:

```
Sortie théâtre
Dans ~4 min
```

---

# 6. Traffic / constraint language

Avoid words like **friction**.

Use:

| Word       | Example            |
| ---------- | ------------------ |
| **Trafic** | Trafic dense       |
| **Bloqué** | Zone bloquée       |
| **Fluide** | Circulation fluide |
| **Pluie**  | Pluie ralentit     |

Example:

```
Pluie forte
Centre actif
```

# 7. Opportunity tone

Instead of saying **“signal”**, use subtle hints.

Examples:

```
Ça bouge Bastille
```

```
Bonne fenêtre
```

```
Moment intéressant
```

But keep it light — not too gamified.

---

# 8. What to completely avoid

These sound like **system diagnostics**:

* signal
* dispersion
* ramification
* saturation
* friction
* champ
* optimisation
* prediction

They break the illusion of **urban intuition**.

Not:

That difference creates **trust and addiction**.
