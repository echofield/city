Good — this is one of the **most important things you can add for mobile**, and it’s not complicated.

Drivers often mount their phone on the dashboard. When they glance at it, they have **less than one second** to read something. That’s why Waze, Uber Driver, and taxi dispatch systems all use **very large simplified views** while driving.

Flow should have something similar: a **Quick Glance Mode**.

---

# 1. What Quick Glance Mode is

It’s a simplified LIVE screen that shows **only the most important information**.

Instead of showing:

* full map
* metrics
* cards
* alternatives

it shows just:

```
MOVE TO OPERA
BD DES CAPUCINES

4 min
STRONG SIGNAL
```

Huge text. Nothing else.

The driver can read it instantly.

---

# 2. When it activates

Two options:

### Option A (best)

Automatically when the phone is **in landscape**.

Drivers often mount phones horizontally.

Example:

```
portrait → normal interface
landscape → glance mode
```

---

### Option B

Toggle button:

```
GLANCE MODE
```

---

# 3. What the screen shows

Only four elements:

### 1️⃣ Command

```
REJOINS OPERA
```

### 2️⃣ Landmark

```
OLYMPIA
```

### 3️⃣ Travel time

```
4 MIN
```

### 4️⃣ Signal strength

```
STRONG
```

Nothing else.

---

# 4. Visual structure

Example layout:

```
REJOINS OPERA

OLYMPIA

4 MIN

STRONG SIGNAL
```

Large typography.

Very high contrast.

---

# 5. Why this matters

Drivers don’t want to interact with the phone constantly.

They want something they can read like a **road sign**.

This makes Flow feel like a **navigation instrument**, not an app.

---

# 6. Gesture interactions

In glance mode you could allow:

```
swipe up → next signal
swipe down → open full interface
tap → navigation
```

Very simple.

---

# 7. When it becomes powerful

During a shift the driver experience becomes:

```
check map
drive
glance screen
follow signal
```

Flow becomes almost like **a co-pilot**.

---

# 8. Why this fits your minimal design

Your design already uses:

* black background
* green signals
* large typography

So glance mode would look extremely clean.

Almost like:

```
military radar interface
```

---

# 9. Future idea

But not necessary now.
