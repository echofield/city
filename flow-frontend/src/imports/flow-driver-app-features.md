# 1️⃣ The “Next Wave” indicator

Right now Flow shows **current signals** very well.

But drivers constantly ask themselves:

> *“Where is the next thing after this?”*

So add a tiny module:

```
PROCHAINE VAGUE
↓
Pigalle — bars
21:15
Courses courtes rapides
```

Example:

```
NEXT WAVE
Pigalle
21:10 – 22:30
Night traffic building
```

Why this is powerful:

Drivers **plan 20–30 minutes ahead**.

So the loop becomes:

```
current signal
↓
execute
↓
next wave
```

This creates **anticipation**, which is addictive.

---

# 2️⃣ The “Driver Density” indicator (very powerful)

Drivers hate going somewhere **where 50 drivers are already waiting**.

So you add a subtle indicator:

Example:

```
CONCURRENCE
LOW
MEDIUM
HIGH
```

Example UI:

```
La Défense
Demand: Moderate
Drivers: LOW
```

or

```
CDG
Demand: High
Drivers: HIGH
```

Drivers instantly think:

```
Less drivers = better odds
```

Even if the estimate is rough.

You can estimate with:

• time of day
• historical data
• signal density
• driver reports later

It **dramatically improves trust**.

---

# 3️⃣ The “Confidence level”

Drivers like knowing **how certain the signal is**.

Add a small label:

```
Confidence: HIGH
```

Example:

```
Stade de France
Sortie match
Score: 85
Confidence: HIGH
```

or

```
Pigalle
Night activity
Score: 65
Confidence: MEDIUM
```

This tells the driver:

```
this signal is strong
```

or

```
maybe
```

Very powerful psychologically.
