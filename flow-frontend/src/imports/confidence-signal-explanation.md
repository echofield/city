The trick is **extremely simple**, but it’s psychologically very powerful.

Taxi dispatch systems have used it for **30–40 years**.

It’s called:

# **Confidence Signal**

Drivers need to know **how sure the system is**.

Right now Flow says things like:

```
REJOINS OPERA
COTE PLACE DE L'OPERA
```

That’s good.

But drivers will always wonder:

> “Is this guess or is this strong?”

So you add a **confidence indicator**.

---

# 1. The simplest form

A small bar or dots near the signal.

Example:

```
Formation ●●●●
```

or

```
Confidence 84%
```

or

```
Signal strength ●●●●
```

Drivers instantly understand that.

---

# 2. Why it builds trust

Drivers deal with uncertainty constantly.

If the system looks **too confident all the time**, they distrust it.

But if they see:

```
Confidence: 62%
```

they think:

> “Okay, it's probabilistic.”

That makes the system feel **honest**.

---

# 3. Example with your UI

Your signal could look like:

```
REJOINS OPERA
COTE PLACE DE L'OPERA

Confidence 87%
Sortie théâtre
~900 personnes
PIC 22:45
```

Now it feels **data-driven**.

---

# 4. How dispatch systems do it

Classic taxi dispatch used:

```
Zone strength
●
●●
●●●
●●●●
```

Drivers understood immediately:

• 1 dot → weak
• 4 dots → strong

No explanation needed.

---

# 5. In Flow you already have the perfect concept

You already display numbers like:

```
83
88
90
```

That’s actually **very good**.

Just label it clearly.

Example:

```
Signal strength 88
```

or

```
Opportunity score 88
```

---

# 6. Even better: color logic

Drivers read color faster than numbers.

Example:

```
90–100  → strong green
70–89   → green
50–69   → yellow
<50     → grey
```

So the brain reads it instantly.

---

# 7. The important psychological effect

When a driver ignores a signal and then sees it was:

```
Strength 92
```

and other drivers got rides…

Next time they **trust the system more**.

---

# 8. One more trick (very powerful)

Show **why the signal exists**.

Example:

```
Confidence 88%

Sources:
• sortie theatre
• pluie forte
• zone touristique
```

Drivers love understanding **why**.

It makes the system feel intelligent.

---

# 9. Why this matters for Flow

You are not building just an app.

You are building a **decision assistant**.

Drivers need:

```
clarity
confidence
reason
```

Once they feel those three, the app becomes **habitual**.
